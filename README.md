# ドローンサウンド by 川越バイオリン教室 — Web 版

耳の師匠を、ポケットに。
バイオリン練習用のドローン（持続音）と、コード進行対応メトロノームのブラウザ版です。

ネイティブ iOS 版（App Store 公開予定）と等価の音律ロジックと加算合成をブラウザで再現しています。

**開く**: https://ngmt4amtk-web.github.io/jawari-web/

## 機能

- **調（主音＋スケール）→ 度数 → ドローン音**の 2 段階選択
- **音律 3 種**: 平均律 / ピタゴラス / 純正律
- **和音構成 6 種**: 単音、1+5、1+8、1+5+8、1+3+5、1+3+5+8
- **音色 5 種**: Pure Sine / Warm Pad / Bright Organ / Tambura / Cello Ensemble
- **基準ピッチ**: A=415 〜 466 Hz
- **メトロノーム & コード進行**: BPM 20〜300（数値入力対応）、1/4・2/4・3/4・4/4・6/4・1/8・3/8、カウントイン（カウントイン中はクリック音のみ）、小節ごとの度数×クオリティ指定
- **カスタム Hz 主音モード**: 任意の基音を Hz で指定し、その上に純正律などで音程を積める
- バックグラウンド再生（OS の制約の範囲内）、Media Session 連携
- 完全ローカル動作（ネットワーク通信なし、localStorage 永続化）

## 音律の設計

本アプリは「曲の調」と「ドローンの度数」を分離して扱います。

- 主音 → ドローン音の関係に、選択中の音律を適用
- ドローン和音の内部（root → 3度 → 5度 → octave）にも同じ音律を適用

例: G 長調 Ⅳ 度の C と、C 長調 Ⅰ 度の C は、純正律では絶対周波数が違います。これは和音の中で本当にハモるバイオリニスト的な音程感覚を再現しています。

## 使い方

1. 円周の 12 音から**主音**を選ぶ
2. 上部の「A 長調」表記タップで**スケール**切替
3. 度数ピル（Ⅰ〜Ⅶ）で**ドローン音**を選ぶ
4. 音律・和音・音色を切り替え
5. 下部の金色再生ボタンで再生

メトロノームは左上アイコンから。

## 技術

- Vanilla HTML / CSS / JavaScript（ビルドなし）
- Web Audio API の `PeriodicWave` で倍音プロファイルをサンプル精度合成
- `setTimeout` ベースのビートスケジューラ
- GitHub Pages で配信
